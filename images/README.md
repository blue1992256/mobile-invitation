## 이미지 최적화 팁

### 파일 크기 줄이기
모바일 성능을 위해 이미지 파일 크기를 최적화하세요:

1. **온라인 도구 사용**
   - [TinyPNG](https://tinypng.com/) - PNG/JPG 압축
   - [Squoosh](https://squoosh.app/) - 구글의 이미지 최적화 도구
   - [Compressor.io](https://compressor.io/) - 무손실 압축

2. **권장 파일 크기**
   - 메인 커버: 200-400KB
   - 프로필 사진: 50-100KB
   - 갤러리 사진: 100-200KB each

3. **포맷 선택**
   - JPG: 사진에 적합
   - PNG: 투명 배경이 필요한 경우
   - WebP: 최신 브라우저에서 더 작은 용량 (선택사항)

### 이미지 비율 맞추기
- **정사각형 (1:1)**: 프로필 사진, 갤러리 사진
- **세로형 (9:16)**: 메인 커버 이미지

## 무료 이미지 소스 (테스트용)

테스트를 위해 무료 이미지를 사용하려면:

- [Unsplash](https://unsplash.com/) - 고품질 무료 사진
- [Pexels](https://www.pexels.com/) - 무료 스톡 사진
- [Pixabay](https://pixabay.com/) - 무료 이미지

검색 키워드: "wedding", "couple", "bride", "groom", "love"
